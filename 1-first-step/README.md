# Primi Passi

-   Navigare fino alla cartella `/1-first-step`
-   Lanciare da terminale `$ node index.js`
-   Oppure `$ npm start`

## Argomenti trattati

Sono riportati e spiegati nel file `index.js`

-   VARIABILI GLOBABLI
-   MODULI CUSTOM
-   MODULO [OS]
-   MODULO [PATH]
-   FILE SYSTEM [SINCRONO]
-   FILE SYSTEM [A-SINCRONO]
-   MODULO HTTP
-   NODEMON
-   EVENT EMITTER
-   STREAM
